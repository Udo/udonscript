# UdonScript Sublime Text Syntax

## Manual install
Copy `UdonScript.sublime-syntax` into your Sublime Text Packages directory:

```bash
mkdir -p "$HOME/.config/sublime-text/Packages/UdonScript"
cp UdonScript.sublime-syntax "$HOME/.config/sublime-text/Packages/UdonScript/"
```

On macOS, use `~/Library/Application Support/Sublime Text/Packages/`.

Restart Sublime; `*.udon` files should highlight automatically.
